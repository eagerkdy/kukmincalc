API 무료 제공

귀사의 웹사이트나 앱에 국민계산기의 35개 계산 기능을 무료로 임베드할 수 있습니다.

  • JavaScript SDK — CalcCore 객체로 직접 함수 호출
  • iframe — 계산기 페이지를 그대로 삽입
  • URL 파라미터 — REST-like 방식으로 JSON 결과 반환

상세한 API 문서는 API 문서 페이지에서 확인하세요.

콘텐츠 활용 안내

유튜브, 블로그, 저서 등 콘텐츠에 국민계산기를 소개하고 활용하고자 하시는 분은 자유롭게 스크린샷을 사용하실 수 있습니다. 출처 표기(kukmincalc.com)만 부탁드립니다.
