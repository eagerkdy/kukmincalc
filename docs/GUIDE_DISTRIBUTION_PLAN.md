## 2. 주차별 배포 한도 (도배 방지)

| 채널 | 일일 한도 | 주간 한도 | 비고 |
|---|---|---|---|
| 네이버 지식인 | 3건 | 15건 | 답변은 항상 결론 → 산식 → 출처 → 링크 순. 같은 글 복붙 금지 |
| 스레드 | 1건 | 5건 | 1~3 슬라이드. 첫 슬라이드 결론 1줄, 마지막 슬라이드 도메인 |
| 네이버 블로그 (요약) | 1건 | 3건 | 요약 + 가이드 백링크 1회 |
| 프리랜서/알바/직장인 카페 | 1건 | 2건 | 카페 규정 확인. 시그니처 도배 금지 |

주차 합산 한도 = 25건. 그 이상 노출은 콘텐츠 품질 저하·신뢰도 손실 위험.

## 5. 7일 후 판단 룰

| 패턴 | 진단 | 액션 |
|---|---|---|
| Impr 충분 + CTR < 1% | title/meta 부적합 | title을 검색 쿼리 그대로 + 결론 수치 노출형으로 수정 |
| Impr 충분 + CTR 정상 + 계산기 CTA 클릭 < 5% | 답변 후 행동 유도 약함 | CTA 위치/문구 강화 (가이드 상단 1회 추가) |
| Impr 충분 + 체류 < 30s | 답변이 결론까지 도달 안 함 | guide-summary를 1줄 결론으로 더 압축 |
| 특정 키워드 CTR > 3% | 검색 의도 매칭 성공 | 유사 키워드 가이드 추가 (Phase 3-4B) |
| Impr 부족 (< 50/주) | 색인 지연 / 의도 미스 | URL 재요청 + 키워드 변형 추가 검토 |

CTR 정상 기준: 위치 1~10 평균 약 3%, 1~3 평균 약 8% (Google 평균치 참고).

## 6. Phase 3-4B 우선순위 후보

7일 측정 후 **반응 높은 키워드 클러스터에서 1~3개 가이드를 추가**하는 후속 PR.

후보 키워드 (검색 의도 검증 대기):
- "연봉 6000 / 7000 / 1억 실수령액" → 클러스터 확장
- "월급 350 / 400 / 500 4대보험" → 4대보험 클러스터 확장
- "주 15시간 주휴수당" / "주 25시간 주휴수당" → 알바 시간대 변형
- "퇴직금 1년 / 5년 / 10년" → 근속별 변형
- "기타소득 100만 / 300만 분리과세 기준" → 8.8% 후속

기준: Week 1~2 상위 3개 키워드에서 파생. 만들기 전 검색량/CPC를 먼저 봐야 함.
