# 2026 기준값 출처 (RATE_SOURCES)

> 모든 요율·한도·정책값은 본 문서의 출처를 우선 인용한다.  
> 민간 블로그·세무사 글·계산기 사이트는 절대 `officialSource` 로 사용 금지 → `referenceSource[]` 에만 사용한다.

`verificationStatus` 정의:
- `verified` — 공식 출처로 검증 완료
- `needs_official_source` — 공식 원문 보강 필요 (계산은 임시값으로 동작 가능)
- `dynamic` — 조건/시점에 따라 변동 (DSR, LTV, 기준금리 등)
- `verified_or_table_based` — 표 lookup 우선 (간이세액표)

## 동적 정책값 (dynamic)

조건/시점에 따라 분기되므로 단일 constants 로 박지 않는다. 입력 조건 기반 `policyTable[]` 또는 formula 로 관리.

| 항목 | 관리 방식 | officialSource | 반영 위치 |
|---|---|---|---|
| DSR 한도 (은행/비은행/스트레스) | `housingLoanPolicy.dsr` (조건 기반) | 금융위, 금감원 | dsr.html (2차 고도화) |
| LTV 한도 | `housingLoanPolicy.ltv.policyTable[]` | 금융위, 금감원 | mortgage-limit.html (2차) |
| DTI 한도 | `housingLoanPolicy.dti.policyTable[]` | 금융위, 금감원 | mortgage-limit.html (2차) |
| 전월세 전환율 상한 | `min(0.10, baseRate + 0.02)` | 한국은행 (기준금리), 주택임대차보호법 시행령 | rent-to-deposit.html (2차) |
| 한국은행 기준금리 | `userEditable: true`, value=null | 한국은행 | 사용자 입력으로 동작 |

## officialSource 사전 (sources-2026.js)

`window.KR_OFFICIAL_SOURCES` 키:

- `mohw` — 보건복지부 (국민연금/건강보험/장기요양)
- `moel` — 고용노동부 (최저임금/주휴수당/퇴직금/산재)
- `nts` — 국세청 (간이세액표/원천징수/연말정산)
- `hometax` — 홈택스
- `wetax` — 위택스 (자동차세/취득세/재산세)
- `mois` — 행정안전부 (지방세법)
- `bok` — 한국은행 (기준금리)
- `fsc` — 금융위원회 (DSR/LTV/DTI 정책)
- `fss` — 금융감독원 (스트레스 DSR 시행 세칙)
- `hf` — 한국주택금융공사 (정책대출)
- `law` — 국가법령정보센터
- `nps`, `nhis`, `kcomwel` — referenceSource 전용 공단

## 갱신 절차

1. 항목별 `lastVerifiedAt` / `nextReviewAt` 도래 시 `assets/js/constants/rates-kr-2026.js` 또는 `policies-kr.js` 수정.
2. `verificationStatus` 가 `needs_official_source` 인 항목이 verified 로 승격되면 `officialSource[]` 에 출처 추가.
3. 본 문서(`docs/RATE_SOURCES.md`) 와 `docs/CHANGELOG-2026-RATES.md` 동시 갱신.
4. `tests/calc-core.test.html` 을 브라우저에서 열어 전 항목 PASS 확인.
