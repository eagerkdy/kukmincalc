## 1. 채널별 루틴

### 1-1. 네이버 지식인

**전략:** 답변형 SEO. 검색량 큰 질문에 정성껏 답변하고 맥락에 맞는 계산기 링크 1회.

**대상 질문 패턴:**
- "연봉 X천 실수령 얼마인가요"
- "주휴수당 발생 조건"
- "프리랜서 3.3% 떼고 얼마 받나요"
- "퇴직금 계산법"
- "DSR 어떻게 계산하나요"

**답변 구조:**
1. 결론 (1~2 문장 + 숫자)
2. 간단한 산식 (1분 안에 따라할 수 있는)
3. 공식 기준 출처 (보건복지부 / 고용노동부 / 국세청)
4. "정확한 계산은 [국민계산기 링크]에서 확인하세요"

**금지:**
- 같은 답변 복붙 도배
- 시그니처에 매번 링크 삽입
- 다른 사이트 비방

### 1-2. 스레드 (Threads)

**전략:** 짧은 시즌 콘텐츠 + 인포그래픽.

**포맷:**
- 1~3개 슬라이드.
- 첫 슬라이드: 결론 1줄.
- 마지막 슬라이드: "더 정확한 계산은 kukmincalc.com" (광고 표현 X).

**시즌 트리거:**
- 1~2월: 연말정산
- 5월: 종합소득세
- 7월: 국민연금 기준소득월액 변경
- 9월·12월: 재산세 / 종부세
- 매월 초: 월급/주휴수당 / 4대보험

### 1-3. 네이버 블로그 (요약 백링크)

**전략:** 가이드 페이지 발행 후 네이버 블로그에 "요약 글 + 원본 링크" 1회.

**포맷:**
- 본문은 요약 (가이드 원본 80% 이내 분량).
- 원본 캐노니컬 링크 1개.
- 자체 인사이트 1~2 문단 추가 (단순 복붙 X).

**금지:**
- 가이드 원문 그대로 복사
- 다수의 블로그에 동일 글 분산 (스팸)
- 광고성 키워드 도배

### 1-4. 카페·커뮤니티

**전략:** 자기 답변 글에서 자연스러운 인용. 별도 글로 광고하지 않음.

**원칙:**
- 카페 질문에 답변하다가 계산기 링크가 정말 필요할 때만 1회.
- 운영자 사전 동의 없이 광고성 글 작성 금지.

### 1-5. 유튜브 (쇼츠 / 롱폼)

**전략:**
- 쇼츠 30~60초: 결론 1개 + 화면 노출 (계산기 결과 캡처).
- 롱폼 5~10분: 가이드 1개 = 영상 1개. 영상 마지막에 캐노니컬 링크.

**대본 소스:** `docs/CONTENT_PRODUCTION_PIPELINE.md` 의 가이드 구조 그대로.

### 1-6. 이미지·인포그래픽

**전략:** og-image 응용 + 시즌별 변형. SNS 공유용.

**규격:**
- 1200×630 (OG)
- 1080×1080 (인스타·스레드)
- 1080×1920 (스토리·쇼츠)

**원칙:** 텍스트 안에 출처 명시 ("보건복지부 2026" 등). 폰트 가독성 우선.

## 2. 발행 후 1주일 루틴

```
Day 0: 가이드 페이지 발행 + sitemap.xml 갱신
Day 0: 네이버 블로그 요약 글 (원본 링크 1개)
Day 1: 네이버 지식인 답변 1~2건 (관련 질문 검색)
Day 2: 스레드 1개
Day 3: 유튜브 쇼츠 1개 (선택)
Day 7: GA4 / Search Console / 이벤트 트래커 데이터 점검
        - 노출수
        - CTR
        - 평균 체류 시간
        - 계산기 클릭률
        - 공식 링크 클릭률
        - 다음 페이지 (related calc)
```

7일 후 결과로 P1 / P2 우선순위 재조정.

## 3. 측정 (Andy Grove)

이벤트 트래커가 발행 콘텐츠로부터 들어오는 모든 행동을 잡는다.

| 이벤트 | 출처 | 의미 |
|---|---|---|
| `calculator_started` | 가이드 → 계산기 링크 클릭 후 첫 입력 | 가이드의 트래픽 → 계산기 전환율 |
| `calculator_completed` | 결과 카드 노출 | 계산 완료율 |
| `related_calculator_clicked` | 결과 후 다른 계산기 카드 클릭 | 체류 / 재유입 |
| `official_link_clicked` | 공식 사이트 카드 클릭 | 신뢰도 / 사용자 다음 행동 |
| `popup_shown` / `popup_cta_clicked` | 결과 팝업 | 수익화 / 공식 링크 노출 |

UTM 파라미터:
- `?utm_source=naver_kin&utm_medium=answer&utm_campaign=salary_3000`
- `?utm_source=threads&utm_medium=organic&utm_campaign=year_end_tax`
- `?utm_source=naver_blog&utm_medium=summary&utm_campaign=...`

## 5. GPTS/Gems 운영 체인 매핑 (Kukmincalc 선별)

> 첨부한 애드센스팜 GPTS/Gems 운영 가이드 중 **외부유입 루틴에 직접 적용되는 것만** 반영.

### 5-1. 지식인 답변 GPT
**용도:** 검색량 큰 질문에 정성 답변 → 가이드/계산기 링크 1회.

**입력 슬롯:**
- 질문 유형 (예: 연봉 4000 실수령 / 프리랜서 3.3%)
- 결론 1줄 + 산식 + 출처 + 계산기 링크

**금지:**
- 답변 복붙 도배
- 시그니처에 매번 링크 자동 삽입

**Phase 3-1 출력:** 본 문서의 §1-1 답변 구조 = 지식인 GPT의 출력 형식 표준.

### 5-2. 스레드 요약봇
**용도:** 가이드/계산기 1편 → 1~3 슬라이드 SNS 카드.

**Kukmincalc 톤:** 결론 1줄 → 산식 1개 → "더 정확한 계산은 kukmincalc.com" (광고 표현 X).

### 5-3. 네이버 블로그 요약봇
**용도:** 원본 가이드의 80% 이내 분량으로 축약 + 자체 인사이트 1~2 문단 + 캐노니컬 링크.

**금지:** 가이드 원문 전체 복사. 한 글 다수 블로그 분산.

### 5-4. 커뮤니티 게릴라 봇 (제한적)
**용도:** 카페/커뮤니티 답변 글에서 자연스러운 인용. **별도 광고 글 작성 X**.

**원칙:**
- 카페 운영자 사전 동의 없이 광고성 글 X.
- 자기 답변 글에서 진짜 도움이 될 때만 1회 인용.

### 5-5. Kukmincalc 에 적용 안 함 (의도적 제외)
- 네이버 애드포스트 승인비서 → Kukmincalc 는 AdSense, 애드포스트 별도 안 씀
- 여행블로그 수익화 비서 → 카테고리 X
- 쿠팡/쇼핑전환 글쓰기 비서 → 쇼핑 카테고리 X (대출비교/세무신고 같은 Kukmincalc-fit 제휴 등장 시에만 부분 참조)
- 실버채널 GPTS / 감성 유튜브 대본 / 사진 프롬프트 전체 → 채널 X

(자세한 제외 목록은 `docs/MULTICHANNEL_EXPANSION_PLAN.md` §제외 대상 참조.)

## 6. Phase 3-1 적용 범위

본 플레이북 자체는 Phase 3-1에서 **정의만 한다**. 실제 외부유입 루틴 가동은 3-2 가이드 페이지 발행 이후.

Phase 3-1에서 즉시 가동하는 것:
- 시즌 섹션 (홈 `.season-section`) → 검색량 시즌 미리 노출
- 공식 사이트 바로가기 → 신뢰도 / 외부 시그널
- og-image / 메타 → 외부 공유 시 일관된 메시지
