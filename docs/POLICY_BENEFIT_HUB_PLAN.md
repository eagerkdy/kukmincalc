## 0. 포지션 (Sun Tzu)

### 우리는 무엇인가
> 공식 사이트로 가기 전에 **내가 대상인지 미리 확인하고, 얼마인지 계산해보는 관문**

### 우리는 무엇이 아닌가
- 정부 사이트 대체 X
- 지원금 신청 대행 X
- 정부 제휴/공식 파트너 X
- "확정 지원금" "100% 환급" 보장 X

### 정부24/보조금24와의 차이
| 정부24/보조금24 | 국민계산기 |
|---|---|
| 공식 신청 채널 | 신청 전 미리 확인하는 도구 |
| 제도 안내가 행정 언어 | 생활 언어 번역 (Sejong) |
| 단순 정보 나열 | 계산기 연결 + 다음 행동 카드 |
| 인증 필요 | 인증 없이 추정 (참고용) |

## 4단 구조 — 모든 혜택 페이지 공통 (Sejong + Ogilvy)

각 혜택 페이지는 정확히 다음 5섹션으로 구성한다. 일관성으로 인지 부하 최소화 + 신뢰 일관성 확보.

```
1. 이게 뭔가요?
   - 한 문장 생활 언어 번역
   - 예: "근로장려금 → 일은 하고 있지만 소득이 낮은 가구에 주는 지원금"

2. 누가 확인해야 하나요?
   - 대상 조건 3~5 줄 bullet
   - "참고 안내, 단정 금지" — "~일 수 있습니다", "공식 기준은 다를 수 있습니다"
   - 단정 표현 절대 금지: "받습니다", "확정", "무조건"

3. 얼마나 받을 수 있나요?
   - 범위 (예: 최대 330만원)
   - 계산 가능하면 → 계산기 CTA 또는 1분 추정 박스
   - "정확한 금액은 홈택스에서 확인하세요" 마무리

4. 어디서 공식 확인하나요?
   - 공식 사이트 카드 1~2개 (label="공식 사이트")
   - target="_blank" rel="noopener noreferrer"
   - 도메인은 공식 기관만 (홈택스/정부24/보조금24/위택스/공단/고용24)

5. 계산기로 먼저 확인할 수 있나요?
   - 해당 시 관련 계산기 카드 1~2개
   - 없으면 섹션 생략 가능
```

각 페이지 추가 요소:
- BreadcrumbList JSON-LD (홈 → 혜택 → 자기)
- FAQPage JSON-LD (4~5 Q&A)
- guide-disclaimer ("참고용입니다" + "공식 기준은 변경될 수 있습니다")

## 5. 홈 진입점 (Aristoteles)

홈 카테고리 카드에 "혜택" 추가. 기존 6 카테고리 + 신규 "혜택" = 7.

```
근로·급여  /  세금  /  부동산  /  대출·금융  /  보험  /  생활  /  혜택 (신규)
```

또는 4축 재정렬:
```
계산  /  점검  /  혜택  /  공식 확인
```

후자가 Phase 4 전체 분류 체계 완성형. 단 기존 사용자 학습 비용 고려하여 **Phase 4B 구현 시 A/B 결정**.
